Scraper API Reference
This API documentation lists all endpoints exposed by the Anime108 Scraper Backend Server. These APIs are used to resolve show info, fetch streaming player URLs, start downloads, and query active status.
GET /search
Searches anime108.com for a query keyword and returns list of matching shows, cover images, and episode count text.
Query Parameters
Param Type Required Description
keyword string No Search query string (alternatively, query using parameter q).
q string No Search query string.
Example Response (200 OK)
[ { "title": "Mushen Ji (Tales of Herding Gods) ตำนานเทพกู้จักรวาล", "url": "https://www.anime108.com/mushen-ji/", "image": "https://www.anime108.com/wp-content/uploads/2024/10/Mushen-Ji.jpg", "episodes_info": "ตอนที่ 1-85" } ]

POST /api/parse
Extracts metadata and episode lists (dubbed/subbed) for a given show page or direct episode page URL.
Request Body
Field Type Required Description
url string Yes The full URL of the Anime108 page (e.g. https://www.anime108.com/mushen-ji/)
Example Response (200 OK)
{ "title": "Mushen Ji (Tales of Herding Gods) ตำนานเทพกู้จักรวาล (พากย์ไทย ซับไทย)", "post_id": 19430, "current_episode": 1, "episodes": { "Thai": [ { "title": "ตอนที่ 1", "url": "https://www.anime108.com/mushen-ji-ep-1/" } ], "Sound Track": [ { "title": "ตอนที่ 1", "url": "https://www.anime108.com/mushen-ji-ep-1/" } ] } }

POST /api/player-url
Resolves the raw embedding player iframe URL for a specific episode page. This is used for previewing streams without downloading full files.
Request Body
Field Type Required Description
url string Yes The URL of the specific episode page.
lang string No Language mode: Sound Track (Subbed, default) or Thai (Dubbed).
Example Response (200 OK)
{ "iframe_url": "https://main.108player.com/index_th.php?id=dc293979261c3a1b852d6e2e" }

POST /api/download
Spawns a background thread to download and merge a video. Returns a unique tracking task_id immediately.
Request Body
Field Type Required Description
url string Yes The target episode page URL to download.
lang string No Language mode: Sound Track (Subbed, default) or Thai (Dubbed).
Example Response (200 OK)
{ "task_id": "c1f7bd8f-f11c-4b5c-a5d6-e9102434da3e" }

GET /api/progress/<task_id>
Polls the active download state, chunk counts, percentage progress, and status log messages for a task.
